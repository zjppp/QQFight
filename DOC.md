

# QQFight

> Q宠大乐斗代玩脚本

## 简介

一款使用python写的Q宠大乐斗游戏代玩脚本。

## 环境依赖

- python 3.7

## 使用指南

1. 下载项目代码

   ```
   git clone https://github.com/zjppp/QQFight.git
   ```

2. 安装依赖

   ```
   pip install -r requirements.txt
   ```

3. 初始化

   复制conf文件夹中的user.conf.example，并重命名为user.conf。

   修改user.conf配置文件

4. 运行

   ```
   py fight.py
   ```
